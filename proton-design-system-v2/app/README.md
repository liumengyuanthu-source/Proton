# Proton Design System App

Interactive React/Vite prototype for the Proton Design System.

## Version

v2.0.0 — June 2026 Component Update

## Update scope

- Kanit typography scale for desktop and mobile
- Primary color palette and neutral scale
- Badge Light component
- Button component with desktop/mobile/small sizes and states
- Input component states
- Dropdown component states and menu behavior
- Horizontal Tabs component
- Updated `proton-design-system.md` documentation

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The prebuilt static version is included in `app/dist`.
